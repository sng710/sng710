1. העלי/החליפי את הקבצים index.html, styles.css, app.js, data.js ו-data.json.

3. תמונות האישיות צריכות להישאר בנתיב:
   images/people-original/person-XXX.jpg

   images/family-groups/...

מה עודכן:
- עיצוב המסך הראשי תואם לגרסת הרפרנס המצולמת.
- התמונות דוהות החוצה ואז נכנסות מחדש בצורה רכה.
- כל אדם מופיע פעם אחת בכל מחזור סיבוב מלא, כדי לשמור על חשיפה שווה.
- החיפוש עוצר את הסיבוב, וכשאין חיפוש הסיבוב חוזר לעבוד.
- תוקנו מצבים שבהם הסיבוב היה עלול לעצור אחרי מעבר ידני/אוטומטי.
